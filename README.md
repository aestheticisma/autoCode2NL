# 基于vscode的代码摘要插件

### 1. Intro

本项目主要包含以下三个部分：

* vscode-extension
* 使用cpp编写的轻量级高性能webserver
  * 用于转发POST请求
  * 作为grpc调用的客户端
* 算法后端
  * 基于CodeBERT的代码摘要生成模型
  * 基于grpc调用的服务端
